```
├─ Home
```

### 2-1. 히어로 섹션 (Hero Section)

**위치**: 페이지 맨 위 (풀스크린 또는 h-screen)

**배경 설정**:
- 베이스 컬러: 블랙 (#000000)
- 텍스처: 마블 이미지 (불투명도 20-30%)
- 그라디언트: 오버레이 (위→아래, 검정→투명→검정)

**콘텐츠**:

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│                                                     │
│         하루 한 집, 깊이를 청소합니다              │  (메인 카피)
│                                                     │  (font-size: 48px~72px)
│                                                     │  (font-weight: light/300)
│                                                     │  (text color: white)
│         고객님의 새로운 시작이 행복과 설렘으로      │
│         가득할 수 있도록, 정성과 책임을 담아        │  (서브 카피)
│         공간을 케어하는 프리미엄 공실 청소          │  (font-size: 16px~20px)
│         서비스입니다.                              │  (text color: light gray)
│                                                     │
│         [ 간편견적문의 ]  [ 유튜브 ]  [ 인스타 ]  │  (버튼 3개)
│                                                     │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**버튼 상세**:

| 버튼 | URL | 색상 | 동작 |

**카드 상세**:

#### 카드 1: 입주청소
```
제목: 입주청소

설명:
신축 시공 과정에서 남은 분진과 잔여 요소를 정리하며,
입주 전 편안하게 머물 수 있는 상태로 공간을 준비합니다.

이런 경우 필요합니다:
• 새롭게 사용될 집이며, 이전 생활 흔적이 없는 상태
• 신축 시공 직후 남은 분진, 타일가루, 실리콘·도배풀 자국이 있는 경우
• 입주 전 공간을 전체적으로 정돈하고 싶은 경우
• 준공 청소가 미흡해 잔여 분진이 남아있는 경우

[자세히 보기] 버튼 → /service 페이지로 이동
```

#### 카드 2: 이사청소
```
제목: 이사청소 (공실 기준)

설명:
이전에 생활이 있었던 공간을 정리하고,
공실 상태에서 다음 사용자가 편안하게 시작할 수 있도록 준비합니다.

이런 경우 필요합니다:
• 이전 사용자가 거주했던 공실 공간
• 퇴거 직후 생활 흔적 정리가 필요한 경우
• 레일, 몰딩, 후드 내부 등 누적 오염 정리가 필요한 경우
• 다음 사용자를 위한 준비 청소가 필요한 경우

[자세히 보기] 버튼 → /service 페이지로 이동
```

#### 카드 3: 인테리어 후 청소
```
제목: 인테리어 후 청소

설명:
시공 과정에서 남은 흔적을 정돈하고,
입주 가능한 컨디션으로 편안히 사용할 수 있도록 준비합니다.

이런 경우 필요합니다:
• 도장·타일·마루 등 시공 완료 이후
• 석고 가루, 실리콘, 접착 흔적이 남아 있는 경우
• 공사 흔적 정리 후 바로 사용 가능한 상태로 만들고 싶은 경우

[자세히 보기] 버튼 → /service 페이지로 이동
```

**카드 특징**:
- 배경: 흰색 또는 밝은 색 (블랙 베이스와 contrast)
- 경계선: 블루 포인트 (상단 또는 좌측)
- 그리드: 반응형 (모바일 1열, 태블릿 2열, 데스크톱 3열)
- 호버 효과: 약간의 shadow 또는 scale up

## 9. 이용 안내 / 필독사항 (Notice) 페이지

### 페이지 구조

**섹션 타이틀**: 이용 안내 / 필독사항

**콘텐츠 형식**: 아코디언 또는 탭 (각 섹션 확장/축소 가능)

### 섹션 1: 작업 제한 및 안내

**제목**: 작업이 불가능하거나 제한되는 경우

**7개 항목**:

```
1. 부분 청소 요청 불가
   부분 청소 요청은 받지 않습니다.
   
2. 시간 제한 청소 불가
   정해진 시간 내 완료를 보장할 수 없으므로,
   시간 제한 청소는 불가능합니다.
   
3. 외창(바깥 유리) 탈거 및 청소 불가
   외부 유리창 청소는 진행하지 않습니다.
   
4. 도배풀 미건조 상태에서의 작업 불가
   최소 2일, 권장 7일 이후 진행합니다.
   
5. 실리콘 절개 및 재시공 작업 불가
   기존 실리콘 제거 및 재시공은 불가능합니다.
   
6. 공사 하자, 스크래치, 유막 제거 등 불가
   청소 범위를 벗어나는 항목입니다.
   
7. 전기 또는 수도 미사용 시 불가
   장비 운용을 위해 전기·수도 사용이 필수입니다.
```

### 섹션 2: 추가 비용 발생 기준

**제목**: 다음의 경우 추가 비용이 발생할 수 있습니다

**6개 항목**:

```
1. 심한 곰팡이
   곰팡이 제거를 위한 특수 처리 필요 시
   
2. 니코틴 타르 변색
   타르 성분으로 인한 변색 제거 필요 시
   
3. 스티커/끈끈이 대량 제거
   대량의 스티커나 접착제 잔여물 제거 필요 시
   
4. 인테리어 후 준공청소 미진행 상태
   시공사 준공청소가 미흡한 상태에서 진행 시
   
5. 실제 평수·구성 차이
   제시된 정보와 실제 상태의 차이 발생 시
   
6. 엘리베이터 미설치/주차 불가 등
   접근 어려움으로 인한 추가 작업 필요 시
```

### 섹션 3: 기본 준비 사항

**제목**: 작업 전 고객이 준비해야 할 사항

**5개 항목**:

```
1. 전기 사용 가능
   장비 운용을 위해 전기 사용이 필수입니다.
   
2. 수도 사용 가능
   청소 과정에서 수도가 반드시 필요합니다.
   
3. 입주 가능한 상태
   작업 전 기본적인 준비 상태가 필요합니다.
   
4. 주차 가능 공간 확보
   작업 팀의 차량 주차 공간이 필요합니다.
   
5. 출입증 필요 시 고객 준비
   보안이 필요한 공간에서는 고객이 출입증을 준비합니다.
```

### 섹션 4: A/S 정책

**제목**: 작업 후 미흡 사항 처리

**내용**:

```
믿고 맡기는 청소는 작업 결과에 끝까지 책임을 갖습니다.

✓ 작업 후 발견된 미흡 사항은 현장 확인 후 조치합니다

✓ 권장 A/S 접수 기한: 작업일 포함 3일 이내

✓ 빠른 재점검과 보완 작업으로 완벽한 마무리를 제공합니다

※ 청소는 사람이 직접 수행하는 작업 특성상
미세한 누락이 있을 수 있습니다.
이 경우 빠른 검토 후 보완 작업을 진행합니다.
(비용 청구 없이 A/S 제공)
```

### 섹션 5: 취소 및 위약금 안내

**제목**: 예약 취소 시 위약금 기준

**5-1. 취소 시 환불 기준**

```
믿고 맡기는 청소는 팀별 1일 1현장 전담제로 운영됩니다.
예약 확정 시 동일 날짜에 다른 예약이 불가능해
취소 기준을 아래와 같이 적용합니다.
```

| 시점 | 기준 | 위약금 | 환불 |
|------|------|--------|------|
| 1 | 작업 7일 전까지 | 0% | 예약금 전액 환불 |
| 2 | 작업 6~4일 전 | 20% | 청소 금액의 80% |

| 4 | 작업 전날~당일 | 50% | 청소 금액의 50% |

**5-2. 현장 상태로 작업 불가 시 (철수 기준)**

```
아래 항목에 해당할 경우 작업이 불가능하며
당일 위약금이 적용됩니다 (총 금액의 50%):

1. 폐기물·자재 등이 정리되지 않아

2. 전기 또는 수도 미개통으로

3. 도배, 페인트, 공사 등 타 공정이
   완료되지 않은 상태
```

**5-3. 현장 진입 불가능한 경우**
